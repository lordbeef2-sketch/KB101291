### Status Codes

The status codes report the status of the operation. To obtain a text 
description of the status code, call 
[niSync_Error_Message](nisync_error_message.html).

The general meaning of the status code is as follows:

| Value | Meaning |
| --- | --- |
| 0 | Success |
| Positive Values | Warnings |
| Negative Values | Errors |
