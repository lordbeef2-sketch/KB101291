### Configuration

Use the Configuration VIs to configure the power-added efficiency measurement.

[IMAGE alt='icon' src='pae-configuration-mnu.png']

- [Configure RF Power Sweep VI](../../../../vi-lib/nipaeml/configure-rf-power-sweep-vi.html) Configures the power range and power step size for the RF power sweep applied to the device under test.
- [Configure Compression Point VI](../../../../vi-lib/nipaeml/configure-compression-point-vi.html) Defines the compression point used for the PxdB measurement. If you are not using PxdB measurement, you do not need to use this VI.
- [Configure DCPower Levels VI](../../../../vi-lib/nipaeml/configure-dcpower-levels-vi.html) Defines the voltage level and the current limit used by the NI-DCPower instrument to excite the external device. In pulse DC mode this specifies the voltage during the on phase of the pulse.
- [Configure DCPower Measurement Interval VI](../../../../vi-lib/nipaeml/configure-dcpower-measurement-interval-vi.html) Specifies the duration of the voltage and current measurement.
- [Configure DCPower Stimulus VI](../../../../vi-lib/nipaeml/configure-dcpower-stimulus-vi.html) Configures the type of stimulus that the DCPower instrument will provide.
- [Configure RFSG Frequency Range Sweep VI](../../../../vi-lib/nipaeml/configure-rfsg-frequency-range-sweep-vi.html) Defines the frequency range used to excite the device under test. It is specified by the start and stop frequencies and the number of points between the range, linearly spaced. Internally it is used to define the NI RFSG frequencies for PAE and PxdB measurements.
- [Configure RFSG Attenuation VI](../../../../vi-lib/nipaeml/configure-rfsg-attenuation-vi.html) Defines the path losses between the RF OUT connector of the signal generator and the input of the device under test.
- [Configure RFSG Stimulus VI](../../../../vi-lib/nipaeml/configure-rfsg-stimulus-vi.html) Configures the type of stimulus generated by RFSG.
- [Configure Expected DUT Gain VI](../../../../vi-lib/nipaeml/configure-expected-dut-gain-vi.html) Configures the expected gain of the device under test. This information will be used to calculate the appropriate level for the acquisition of the transmission power.
- [Configure RFmx Frequency Range Sweep VI](../../../../vi-lib/nipaeml/configure-rfmx-frequency-range-sweep-vi.html) Defines the frequency range used to measure the device under test. Internally it specifies the start and end frequency for the NI RFmx instrument, and uses the same number of steps as the Configure RFSG Frequency Range Sweep VI.
- [Configure RFmx Attenuation VI](../../../../vi-lib/nipaeml/configure-rfmx-attenuation-vi.html) Defines the path losses between the RF IN connector of the signal analyzer and the device under test.
- [Configure RFmx Measurement Interval VI](../../../../vi-lib/nipaeml/configure-rfmx-measurement-interval-vi.html) Specifies the acquisition time, in seconds, for the RF power measurement.
- [Configure RFmx RBW Parameters VI](../../../../vi-lib/nipaeml/configure-rfmx-rbw-parameters-vi.html) Defines the Resolution Bandwidth (RBW) filter to measure the RF power of the signal as seen through this filter. Note: the RRC Alpha input is only applicable when RBW Filter Type is RRC .
- [Configure DCPower to RFSG Trigger VI](../../../../vi-lib/nipaeml/configure-dcpower-to-rfsg-trigger-vi.html) Configures the trigger that synchronizes the RF pulse to the DC pulse.
- [Configure RFSG to RFmx Trigger VI](../../../../vi-lib/nipaeml/configure-rfsg-to-rfmx-trigger-vi.html) Configures the trigger that synchronizes the RF measurement with the RF pulse generation.
- [Configure RFmx to DCPower Trigger VI](../../../../vi-lib/nipaeml/configure-rfmx-to-dcpower-trigger-vi.html) Configures the trigger that synchronizes the DCPower measurement to the RFmx measurement.
- [Configure DCPower Pulse Trigger Input VI](../../../../vi-lib/nipaeml/configure-dcpower-pulse-trigger-input-vi.html) Defines the trigger that the DCPower instrument will wait for before generating each pulse.

- [Get Frequency for Iteration VI](../../../../vi-lib/nipaeml/get-frequency-for-iteration-vi.html) Gets the specific frequency that you apply in the iteration based on the index value. The index value is specified based on the calculations of the power and frequency sweep.
- [Save PAE Results VI](../../../../vi-lib/nipaeml/save-pae-results-vi.html) Save PAE results into a TDMS file.
- [Load PAE Results VI](../../../../vi-lib/nipaeml/load-pae-results-vi.html) Load PAE results from a tdms file.
