LabVIEW NXG 5.1 Real-Time Module
 Manual

Use the LabVIEW NXG Real-Time Module to design a real-time application, implement
 your real-time code, and deploy the application to a real-time target. This manual includes
 programming concepts, step-by-step instructions, and reference information about using the
 LabVIEW NXG Real-Time Module.

#### Top Tasks

- Design a
 real-time application —Plan the structure of your application before you
 begin writing code to help you make effective programming decisions for your application.
- Implement
 a real-time application design —Create and execute code using programming
 practices specific to real-time applications.
- Test and deploy a real-time application to a target —Use interactive
 execution and benchmarking tools to test the determinism of your application and then
 deploy the application to the real-time controller.

#### Programming Reference

- Programming with a Real-Time Target
