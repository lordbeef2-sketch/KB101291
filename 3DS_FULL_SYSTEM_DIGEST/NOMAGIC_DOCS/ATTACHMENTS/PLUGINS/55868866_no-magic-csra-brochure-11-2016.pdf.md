Project Templates

There are two templates predefined in the plugin:

   • FMEA Project (Failure Mode Effects Analysis Project). Select this template if you need the reliability
     analysis only.

   • Safety and Reliability Analysis Project (FMEA Project included). Select this template if you need
     both the FMEA and risk analysis.

Both templates contain predefined packages and diagrams to start performing risk analysis. Depending
on the template, Design, Reliability Analysis, Risk Analysis and Safety Requirements top-level packages
are created, followed by appropriate sub packages.
