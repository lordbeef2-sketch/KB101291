- release date: June 26, 2026

### MagicDraw / Cameo Developer Guide 2024x

Official developer guide reports:

- generated JavaDocs are available online and under <modeling tool installation directory>/openapi/docs

- core code/libraries are under <modeling tool installation directory>/lib

- plugin libraries can be under <modeling tool installation directory>/plugins/<plugin_name>

- API scope annotations distinguish OpenAPI, InternalAPI, and NotAPI

- only OpenAPI scope should be used for extending the tool

### Model editing docs

Official docs report:

- ElementsFactory creates model elements

- SessionManager sessions are required for creating/editing elements

- closing a session registers changes in command history

- only one session can be active

### Stereotype/tag docs

Official docs report:

- use StereotypesHelper / TagsHelper

- prefer profile URI + Stereotype object instead of global name-only stereotype matching

- tagged-value helpers may return java.util.List even for single-valued properties

- SysML v1.7 was published June 2024

- SysML v2.0 formal specifications are listed as September 2025

- Systems-Modeling SysML v2 release repository says the specifications were formally adopted by OMG as of June 30, 2025 and editorially updated in March 2026 for ISO submission

- latest GitHub release shown: 2026-04 - SysML v2 Release on May 14, 2026
