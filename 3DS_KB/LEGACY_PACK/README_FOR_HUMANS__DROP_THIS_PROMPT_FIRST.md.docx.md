# README for Humans — Drop This Prompt First

Snapshot date: 2026-07-09

## What this folder is

This `3DS` folder is an AI-ready knowledge pack for Dassault / No Magic / CATIA Magic work around:

- Cameo Systems Modeler / MagicDraw / Magic Cyber Systems Engineer

- Teamwork Cloud / Magic Collaboration Studio

- SysML v1 and SysML v2

- Cameo OpenAPI plugin development

- model reading and safe model editing

- TWC REST / OSLC / simulation / SysML v2 service workflows

- Cameo Simulation Toolkit

- Cameo DataHub source routing

- EICD / ICD / pinout workflows

This is not a magic replacement for official vendor documentation, licensed local JavaDocs, or testing against a real model. It is a source-routing and operating library so an AI knows where to look, what surface to use, and what not to guess.

## Human instructions

When giving this folder to an AI:

1. Upload or provide access to the entire `3DS` folder.

2. Paste the prompt below first.

3. Tell the AI the actual target version: 2022x, 2024x Refresh3, 2026x, or 2026x Refresh1.

4. Tell the AI the task type: plugin, report, TWC API, OSLC, simulation, SysML v2, DataHub, EICD/pinout, or migration.

5. For production code, provide or verify the local installed JavaDocs for the exact target version.

6. Never run mutating AI-generated code directly on a production model first. Use a test model and dry-run output.
