### NORMALIZED CONTENT

MagicLab Collaborator allows you to provide contextual feedback on SysML v2 elements through several dedicated commenting features.

##### Comments tab in Specification panel

You can view and create new comments or replies directly from the Specification panel. A dedicated Comments tab shows only the comments associated with the currently selected element.

To open the Comments tab

1. Double-click an element to open its Specification panel.
2. At the top of the Specification panel, select the Comments tab. [ATTACHMENT filename='comments_in_specification_panel.png']

###### Working with the Comments tab

[IMAGE alt='' src='']

###### Viewing the Comments tab in the Specification panel.

In the Comments tab, you can:

- Create a new comment in the New comment section.
- In the **Comments**section, you can:
  - Filter resolved and SysML v2 comments. The SysML v2 comments only include the title and body text with no additional metadata, such as an author or date created.
  - Search for comments.
  - Select a comment to view the menu [IMAGE alt='' src='']
    - In this menu, you can select to **Edit** the comment or **Delete conversation**to delete the entire comment thread.

##### Project Comments panel

You can view all project comments in a dedicated Project comments panel. To access it, click its icon in the toolbar:

[IMAGE alt='' src='']

###### Viewing Project comments panel.

- In the **Project comments**panel, you can:
  - Filter resolved and SysML v2 comments.
  - Search for comments.
  - Select a comment to view the menus [IMAGE alt='' src='']:
    - The upper menu on the right allows you to edit the comment or delete the entire comment thread
    - The lower menu, displayed in the related element bar, allows you to select the comment in the Containment tree.
  - Click [ATTACHMENT filename='navigate.png'] to navigate to the comment with the related element.

##### Managing licenses via Teamwork Cloud Admin

MagicLab Collaborator uses Cameo Collaborator licenses. To apply a Cameo Collaborator license, see [CONFLUENCE_PAGE title='Cameo Collaborator for Teamwork Cloud licensing' space='MCS'].

###### Applying a license via Teamwork Cloud Admin

To apply a MagicLab Collaborator license via Teamwork Cloud Admin

1. Open Teamwork Cloud in a web browser using this address format: http(s)://<server_address>:<port>/webapp . 8443 is the default port.
2. Sign in to Teamwork Cloud with a user who has the Server Administrator role .
3. Click the Settings application to open it.
4. From the left-side menu, select the MagicLab Collaborator license page. [ATTACHMENT filename='magiclab_collaborator_license.png']
5. Select either the FlexNet or DSLS tab:
  - In the FlexNet tab, enter the license server address to view the list of available licenses. [ATTACHMENT filename='magiclab_collaborator_flex.png'] Select a license and click Confirm to apply.
  - In the DSLS tab, select either Organization defined or Custom.
    - If you select Organization defined, choose a license from the list and click Confirm to apply.
    - If you select Custom, enter the license server address, choose a license from the list and click Confirm to apply.

### NORMALIZED CONTENT

To open a SysML v2 resource in MagicLab Collaborator

1. Go to the Resources application.
2. To open the latest version in the project trunk :
  1. Click [ATTACHMENT filename='actions.png'] next to the resource you want to open and select Open in MagicLab Collaborator . [ATTACHMENT filename='open_in_mlab_collab.png']
3. To open the latest version of a branch :
  - Open the Resource details.
  - In the Branches card of the Resource details panel. click [ATTACHMENT filename='actions.png'] next to the branch you want to open and select Open in MagicLab Collaborator . [ATTACHMENT filename='open_branch.png']
4. To open a historical version:
  - Click View all at the bottom of the History card.
  - Click [ATTACHMENT filename='actions.png'] next to the version you want to open and select Open in MagicLab Collaborator. 
[IMAGE alt='' src='']

MagicLab Collaborator allows you to provide contextual feedback on SysML v2 elements through several dedicated commenting features.

##### Comments tab in Specification panel

You can view and create new comments or replies directly from the Specification panel. A dedicated Comments tab shows only the comments associated with the currently selected element.

To open the Comments tab

1. Double-click an element to open its Specification panel.
2. At the top of the Specification panel, select the Comments tab. [ATTACHMENT filename='comments_in_specification_panel.png']

###### Working with the Comments tab

[IMAGE alt='' src='']

###### Viewing the Comments tab in the Specification panel.

In the Comments tab, you can:

- Create a new comment in the New comment section.
- In the **Comments**section, you can:
  - Filter resolved and SysML v2 comments. The SysML v2 comments only include the title and body text with no additional metadata, such as an author or date created.
  - Search for comments.
  - Select a comment to view the menu [IMAGE alt='' src='']
    - In this menu, you can select to **Edit** the comment or **Delete conversation**to delete the entire comment thread.

##### Project Comments panel

You can view all project comments in a dedicated Project comments panel. To access it, click its icon in the toolbar:

[IMAGE alt='' src='']

###### Viewing Project comments panel.

- In the **Project comments**panel, you can:
  - Filter resolved and SysML v2 comments.
  - Search for comments.
  - Select a comment to view the menus [IMAGE alt='' src='']:
    - The upper menu on the right allows you to edit the comment or delete the entire comment thread
    - The lower menu, displayed in the related element bar, allows you to select the comment in the Containment tree.
  - Click [ATTACHMENT filename='navigate.png'] to navigate to the comment with the related element.

##### Managing licenses via Teamwork Cloud Admin

MagicLab Collaborator uses Cameo Collaborator licenses. To apply a Cameo Collaborator license, see [CONFLUENCE_PAGE title='Cameo Collaborator for Teamwork Cloud licensing' space='MCS'].

###### Applying a license via Teamwork Cloud Admin

To apply a MagicLab Collaborator license via Teamwork Cloud Admin

1. Open Teamwork Cloud in a web browser using this address format: http(s)://<server_address>:<port>/webapp . 8443 is the default port.
2. Sign in to Teamwork Cloud with a user who has the Server Administrator role .
3. Click the Settings application to open it.
4. From the left-side menu, select the MagicLab Collaborator license page. [ATTACHMENT filename='magiclab_collaborator_license.png']
5. Select either the FlexNet or DSLS tab:
  - In the FlexNet tab, enter the license server address to view the list of available licenses. [ATTACHMENT filename='magiclab_collaborator_flex.png'] Select a license and click Confirm to apply.
  - In the DSLS tab, select either Organization defined or Custom.
    - If you select Organization defined, choose a license from the list and click Confirm to apply.
    - If you select Custom, enter the license server address, choose a license from the list and click Confirm to apply.
