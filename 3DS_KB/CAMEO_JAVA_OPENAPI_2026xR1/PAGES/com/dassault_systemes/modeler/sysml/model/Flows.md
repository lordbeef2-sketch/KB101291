Utility methods for identifying flow-related types within the model.
 A flow type may appear either as a [`FlowUsage`](sysml/FlowUsage.html) or a [`FlowDefinition`](sysml/FlowDefinition.html).
 This helper provides a simple predicate for checking whether a given [`Type`](../../kerml/model/kerml/Type.html)
 represents either form.
