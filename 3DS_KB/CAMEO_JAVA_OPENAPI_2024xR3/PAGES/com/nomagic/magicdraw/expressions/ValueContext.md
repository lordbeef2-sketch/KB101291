@OpenApiAllpublic classValueContext

Value context is used when getting value with [`ParameterizedExpression`](ParameterizedExpression.html).
 The invoker of the [`ParameterizedExpression`](ParameterizedExpression.html) passes the actual value context instance,
 while the expression implementation is responsible if to react according the context instance.
