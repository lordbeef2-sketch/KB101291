KerML
—
Kernel Modeling Language
The Kernel Modeling Language (KerML) is an application-independent modeling language with a well-grounded formal semantics for modeling existing or planned systems. The language includes general syntactic constructs for structuring models, such as relationships, annotations and namespaces; core semantic constructs that have semantics based on classification; and additional constructs for commonly needed modeling capabilities, such as associations and behaviors.

<h2>
<span class="spec-acronym" style="text-transform: none">KerML</span>
                     — 
                    <span class="spec-name" style="text-transform: none">Kernel Modeling Language</span>
</h2>
</div>
<div class="container">
<div class="col-md-11">
<p class="text-justify">The Kernel Modeling Language (KerML) is an application-independent modeling language with a well-grounded formal semantics for modeling existing or planned systems. The language includes general syntactic constructs for structuring models, such as relationships, annotations and namespaces; core semantic constructs that have semantics based on classification; and additional constructs for commonly needed modeling capabilities, such as associations and behaviors.</p>
