begin-user-doc 
 A representation of the model object '***Association Class***'.
 end-user-doc 
begin-model-doc 
 A model element that has both Association and Class properties. An AssociationClass can be seen as an Association that also has Class properties, or as a Class that
 also has Association properties. It not only connects a set of Classifiers but also defines a set of Features that belong to the Association itself and not to any
 of the associated Classifiers.
